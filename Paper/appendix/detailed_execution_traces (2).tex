\section{Detailed Execution Traces}
\label{app:detailed_traces}

This appendix provides complete parameter and output records for the worked examples presented in Sections~\ref{subsubsec:worked_example_case_I} (Case I) and~\ref{subsec:saab_case} (Case II). These tables demonstrate how each function invocation materializes concrete artifacts in \texttt{FunctionOutputInSession} and \texttt{FunctionParametersInSession}, enabling fine-grained inspection, replay, and failure attribution at record level.
